== Description ==

Adds a promotional product to the cart when certain conditions are met

== Installation ==

Upload the Gift to cart plugin to your site, Activate it.
To use the Gift to cart, simply install and select the settings.
The plugin has two options.
First: when the value of the basket exceeds the specified amount
Second: when the customer has selected a specified (one of the specified) product (-s).

== Frequently Asked Questions ==

= 1. How can I use the plugin? =
1. Create a new category (only for promotional products).
2.Copy the promotional item that we add to the cart and create a new one in which to change "Visibility in the catalog" to "Hidden". Choose the category of item that you created for the promotion.
3. In the settings, set the background color(e.g #d9b37b), text color, link color, promotional category, show popup checkbox, popup width and remove data from database when plugin is removed checkbox.
